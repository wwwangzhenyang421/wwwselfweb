# Wang Zhenyang - Personal Portfolio

个人作品集网站，基于 React 开发。

## 关于

这是 Wang Zhenyang 的个人作品集网站，展示个人项目、技能和联系方式。

## 本地开发

### 安装依赖

```shell
yarn install
```

### 启动开发服务器

```shell
yarn start
```

项目将在 [http://localhost:3000](http://localhost:3000) 运行。

### 构建项目

```shell
yarn build
```
